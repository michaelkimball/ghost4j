Ghost4J is a Java wrapper for the Ghostscript API, and more.

[Ghostscript](http://www.ghostscript.com) is a free, powerful, and widely used PostScript language (and PDF) interpreter available on multiple platforms. However, it only provides a C API with no Java bindings.

Ghost4J bridges that gap using JNA, making the full Ghostscript API available in Java.

## Main Features

- Full Ghostscript API supported via JNA
- Object-oriented API
- High-level API with components: converters, analyzers, modifiers, and renderers for PostScript and PDF documents
- Multi-process support for concurrent environments
